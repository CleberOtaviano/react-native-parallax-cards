## Running locally with exp CLI
### Grab the repo
```shell
$ git clone https://github.com/zachgibson/ParallaxCardsDemo.git && cd ParallaxCardsDemo
```

### Install packages
```shell
$ yarn install
```
or
```shell
$ npm install
```

### Start your packager
```shell
$ exp start
```

### Install the app
```shell
$ exp ios
```
Or scan the QR code generated by the CLI to run on device
